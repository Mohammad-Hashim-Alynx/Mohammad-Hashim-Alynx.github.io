Advanced Smart Pool Water Quality Monitoring System

Production-grade IoT system for continuous, automated monitoring and management of swimming pool water chemistry.

Project Overview

This project is a production-grade IoT system designed to provide continuous, real-time monitoring of swimming pool water quality. It addresses the real-world challenges of inconsistent water chemistry, excessive chemical usage, and labor-intensive manual testing commonly found in residential and commercial pool maintenance.

Unlike periodic manual testing using strips or handheld meters, the system delivers 24/7 sensor-driven data, precise chemical adjustment calculations, and automated alerts. It eliminates guesswork, reduces chemical waste, protects pool infrastructure, and ensures a consistently safe and comfortable swimming environment.

System Architecture

The system follows a layered, modular architecture centered around an Arduino-based controller (Arduino Uno with expanded I/O). Dedicated sensor subsystems handle TDS/salinity, pH, and temperature, each with appropriate signal conditioning and calibration logic.

Hardware Components

The system uses high-quality industrial sensors for reliable long-term operation in pool environments, with appropriate signal conditioning and data processing.

Data Processing & Control Logic

The firmware is built around a non-blocking finite state machine (FSM) that manages monitoring, menu navigation, calibration routines, and log viewing without halting system operation.

Sensor data is sampled at fixed intervals using millis()-based timing, filtered using moving averages, validated against physical limits, and compensated for temperature effects before being displayed, logged, or transmitted.

Chemical Calculation Engine

The system translates raw sensor data into actionable maintenance instructions. For saltwater pools, it calculates the exact mass of salt required based on current salinity, target salinity, and configured pool volume.

pH adjustment logic estimates the quantity of acid or base needed to reach target pH levels, warning users to apply chemicals gradually. This precision prevents chemical over-dosing, infrastructure damage, and unsafe swimming conditions.

Firmware Architecture & Reliability

The firmware is modular and maintainable, with dedicated modules for sensors, calculations, storage, communication, and user interface. Calibration data and system configuration are preserved in EEPROM, while SD card logging provides long-term historical records.

Watchdog timers, sensor validation checks, and fail-safe defaults ensure stable 24/7 operation even under partial hardware or communication failures.
