Alynx™ Geolocation-Based NTP WiFi LCD Clock

ESP32-based internet-synchronized digital clock with automatic timezone handling, large-format display, and web-based configuration.

Project Overview

The Alynx™ NTP WiFi Clock is a professional-grade timekeeping system designed to deliver accurate, unattended time synchronization using internet NTP servers. It eliminates time drift, manual setting, and post-power-loss reconfiguration commonly seen in conventional digital clocks.

The system automatically connects to WiFi, synchronizes time via NTP, applies geographic timezone offsets, and presents the information on a large, high-visibility 20×4 LCD using custom-rendered digits.

System Architecture

The system is built around an ESP32 microcontroller, selected for its integrated WiFi, dual-core processing, and reliable real-time performance. Time is obtained from global NTP servers and corrected using configurable timezone offsets, including fractional offsets such as UTC+5:30.

A 20×4 I²C LCD provides a large-format visual interface, while an embedded web server enables configuration without firmware reflashing.

Key Features

  • Automatic NTP-based time synchronization
  • Large custom-rendered digits for high readability
  • 12-hour and 24-hour display modes
  • Automatic timezone and offset handling
  • Web-based configuration interface
  • Persistent configuration storage
  • Robust WiFi auto-reconnect logic

Connectivity & Reliability Engineering

The firmware implements multi-stage WiFi reconnection logic optimized for real-world power and network instability. After power restoration, the system aggressively retries connections during early boot and transitions to stable retry intervals during normal operation.

During internet outages, the clock continues running using the ESP32’s internal RTC and automatically resynchronizes once connectivity is restored, ensuring minimal drift and uninterrupted display.

User Interface & Configuration

A built-in web interface allows configuration of WiFi credentials, timezone offsets, and display format using any standard web browser. A physical push-button provides manual access to configuration mode and recovery in case of network changes.

The display provides clear system feedback, including connection status, date, and synchronization state, enabling unattended long-term deployment.

Applications

  • Server rooms and IT facilities
  • Classrooms and laboratories
  • Industrial control rooms
  • Public display and reception areas
