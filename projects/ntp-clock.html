Project Overview

The Alynxâ„¢ NTP WiFi Clock is a professional-grade timekeeping system designed to deliver accurate, unattended time synchronization using internet NTP servers. It eliminates time drift, manual setting, and post-power-loss reconfiguration commonly seen in conventional digital clocks.

The system automatically connects to WiFi, synchronizes time via NTP, applies geographic timezone offsets, and presents the information on a large, high-visibility 20Ã—4 LCD using custom-rendered digits for maximum readability.

System Architecture & Components

Hardware Components

The system uses an ESP32 microcontroller for WiFi connectivity and processing, paired with a 20x4 LCD display for large-format time presentation.

Wiring & Connections

Clean I2C-based wiring minimizes connections while ensuring reliable communication between the ESP32 and LCD display.

Display Layout & Interface

The 20x4 LCD is optimized for maximum readability with custom large digits and clear status information organized across four rows.

WiFi Connectivity & Reconnection Strategy

The firmware implements multi-stage WiFi reconnection logic optimized for real-world power and network instability. After power restoration, the system aggressively retries connections during early boot and transitions to stable retry intervals during normal operation.

Time Synchronization & NTP Handling

NTP synchronization ensures millisecond accuracy, while the ESP32's internal RTC maintains time during network outages with minimal drift.

Web Configuration Portal

Built-in web interface allows easy configuration without firmware reflashing. Accessible via any web browser on the local network.

Button Functions & User Interaction

Single button provides multiple functions based on press duration, enabling configuration mode access and manual reconnection.

Configuration Settings

All system parameters are configurable through the web interface, with sensible defaults for immediate operation.
