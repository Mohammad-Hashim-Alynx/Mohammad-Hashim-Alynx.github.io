Alynx BLE HID Gamepad Controller

Cross-platform, low-latency Bluetooth Low Energy (BLE) HID gamepad designed for universal compatibility and reliable wireless input.

Project Overview

Alynx is a standards-compliant BLE HID gamepad controller designed to function as a native input device across Windows, Linux, macOS, Android, and iOS without custom drivers. It addresses limitations of proprietary or wired controllers by offering low latency, low power consumption, and flexible firmware-based customization.

The controller implements the HID over GATT (HoG) profile, allowing it to be instantly recognized by host operating systems as a generic game controller suitable for gaming, robotics teleoperation, media control, and accessibility interfaces.

System Architecture & Components

Hardware Components

The system is built around the XIAO ESP32-C6 microcontroller, which provides native Bluetooth 5.0 Low Energy support and efficient RISC-V processing. Physical button inputs are sampled via GPIO, encoded into compact HID input reports, and transmitted wirelessly to the host using BLE notifications.

Pin Configuration & Wiring

Seven digital buttons are connected to ESP32-C6 GPIOs configured with internal pull-up resistors. Buttons pull the input low when pressed. Software debouncing ensures reliable state detection without false triggers caused by mechanical bounce.

BLE Protocol & HID Implementation

Alynx implements a fully compliant BLE HID device using the HID over GATT profile. The controller advertises the Human Interface Device service and exposes the required characteristics for HID information, report mapping, control, and input reporting.

Button Mapping & Gaming Applications

Input reports consist of a compact three-byte structure representing seven digital buttons and two centered axes, minimizing BLE packet overhead and ensuring low-latency input delivery.
